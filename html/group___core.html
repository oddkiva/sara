Detailed Description

Note that the Core module heavily relies on the Eigen library (linear algebra).

the Eigen namespace is directly imported inside the DO namespace for convenience. For details, have a look at the file EigenExtension.hpp.

So more specific information about the Eigen library, refer to: http://eigen.tuxfamily.org/dox/
